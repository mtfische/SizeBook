MainActivity.class:
-main activity class where most user activity takes place
-implements a list of people to display sies
-edit button links to edit user activity
-delete button will remove person
-add button links to add user activity

AddUser.class:
-provides a form field to imput user data
-100% verification of fields with error message reporting
-add button creates person using form data
-cancel button finishes activity

EditUser.class:
-provides a form field to imput user data
-100% verification of fields with error message reporting
-edit button replaces person data with form data
-cancel button finishes activity

IOhandler.class:
-handles all save load functions

PersonAdapter.class:
-inflates a listview with person objects 
-adds edit & delete buttons
